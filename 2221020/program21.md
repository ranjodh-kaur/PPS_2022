 ## Program 21: Write a program illustrating the use of float and double.(Tutorial 2)
 ```C
#include<stdio.h>
int main()
{
float mark1= 94.678;
double mark2= 91.32;
printf("Niko scored= %f", mark1);
printf("\nCassie scored= %lf",mark2);
return 0;
}
```
```
Output:Niko scored= 94.678001
Cassie scored= 91.320000
```

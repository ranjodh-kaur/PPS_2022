## Program 3: Write a program to get user input
```c
#include<stdio.h>

int main(){
	char usr[20];
	printf("Enter a username : ");
	scanf("%s",&usr);
  int rollno;
  printf("Enter roll number: ");
  scanf("%d",&rollno);
  
  printf(" Username = %s",usr);
  printf("Roll number = %d",rollno);

  return 0;
}
```
### Output:
```
Enter a username : Ajay441
Enter roll number: 2221299
 Username = Ajay441
Roll number = 2221299
```

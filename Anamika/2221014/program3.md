## program 3: write a program to input two numbers using scanf() and add them 

#include <stdio.h>

int main()

{

int a, b, sum;

printf("enter first number-");

scanf("%d",&a);

printf("enter second number-");

scanf("%d",&b);

sum = a + b;

printf("%d + %d = %d", a,b, sum ); 

return 0; 

}

**OUTPUT : Enter first number- 6**

**enter second number- 7**

**6+7=13**

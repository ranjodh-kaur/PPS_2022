## Program 8 : Write a program to use switch statement

```C
#include<stdio.h>
int main()
{
int day = 4;
switch(day)
{
case 6:
printf("Today is Saturday");
break;
case 7:
printf("Today is Sunday");
break;
default:
printf("Looking for next weekend");
}
return 0 ;
}
```

## Output : Looking for next weekend

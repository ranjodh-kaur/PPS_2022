## Program 16 : Write a program  to convert ascii value to chracter.
```C
#include<stdio.h>
int main()
{
    int ascii;
    printf("Enter the ASCII value");
    scanf("\n%d",&ascii);
    printf("\nascii to char=%c",ascii);
    return 0;
}
```

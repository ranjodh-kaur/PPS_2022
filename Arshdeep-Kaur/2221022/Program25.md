## Program 25: Write a program to reverse a string

#include<stdio.h>

#include<string.h>

int main()

{

	char a[25]="ARSHDEEP";
  
	printf("The given string is = %s",a);
  
	printf("\nAfter reversing string is = %s",strrev(a));
  
	return 0;
  
}

**Output :The given string is = ARSHDEEP**

**After reversing string is = PEEDHSRA**
